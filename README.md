# AI Interview Simulator

A local web app that simulates a timed interview using AI-generated questions, AI voice, and speech-to-text answers.

## Features
- Timed interview sessions
- AI-generated interviewer questions
- AI-generated interviewer voice (TTS)
- Browser speech-to-text for answers
- Adaptive follow-up questions
- Transcript with feedback and weaknesses
- Final result with confidence score

## Tech Stack
- Frontend: HTML, CSS, JavaScript (Web Speech API)
- Backend: Node.js, Express
- OpenAI API: Chat Completions and Text-to-Speech
